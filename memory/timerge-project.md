timerge = 회사 근무시간을 기록·계산하고 **남은 목표를 역산**해주는 앱 (FLEX 근태앱 캡처를 입력으로). 상세 설계는 프로젝트 루트 `DESIGN.md`에 있음 — 작업 전 반드시 참고.

**스택**: React + Vite + TypeScript, Capacitor(모바일 스토어 출시), 로컬 IndexedDB(Dexie). 동기화는 2단계로 분리(1단계는 기기 로컬 only + JSON 백업). 데스크톱 웹 + 모바일 둘 다 한 코드베이스.

**핵심 계산 규칙 (비자명, 함정 주의)**:
- 모든 시간 **정수 분(minute)** 저장, 표시만 포맷. 반올림 없음.
- 계산의 단일 진실원천 = 각 날 `recognizedMinutes`. 출퇴근 시각은 메타데이터(표시·역산 입력용).
- 점심 60분 무조건 1회 차감. **단 오전 반차/연차가 있는 날은 추가 근무에 차감 안 함**(오전 휴가 자리비움에 점심 포함). 판정: 휴가 구간이 모든 근무 구간보다 시간상 앞이면 면제.
- 주 목표 `goal = baseGoal(디폴트 40h) − 공휴일수×8h`. 공휴일 = 목표 −8h + 누적 0(`0:00` 표시, FLEX와 동일). 연차=480·반차=240분은 누적 기여.
- ⚠️ **double-subtract 금지**: 공휴일에 목표를 줄이면서 동시에 8h를 누적에 더하면 안 됨(공휴일 1주에 6h/일 버그). "목표 −8 + 누적 0" 한 방식으로만.
- 초과 = max(0, 누적 − goal). 주말 근무는 누적에 포함(목표 먼저 채우고 초과로 넘침).
- 마지막 근무일 퇴근 = 출근 + 필요시간 + 점심. "마지막 근무일"은 요일 하드코딩 아님(그 주 마지막 근무가능일).
